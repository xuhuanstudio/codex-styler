# Support

Codex Styler is a community-maintained, unofficial project.

- Installation, compatibility, theme import, and restore problems: use GitHub Issues.
- Usage questions and theme ideas: use GitHub Discussions after it is enabled for the public Beta.
- Security problems: use private vulnerability reporting.
- OpenAI Codex product or account problems: use official OpenAI support, not this repository.

Before filing an issue, try Restore official, restart Codex normally, and check [COMPATIBILITY.md](COMPATIBILITY.md).
