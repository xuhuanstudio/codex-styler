# Support

Codex Styler is a community-maintained, unofficial project.

- Installation, compatibility, theme import, and restore problems: use GitHub Issues.
- Usage questions and theme ideas: use GitHub Discussions after it is enabled for the public Beta.
- Security problems: use private vulnerability reporting.
- OpenAI Codex product or account problems: use official OpenAI support, not this repository.

Before filing an issue, try **Restore official**, restart Codex normally, and check [COMPATIBILITY.md](COMPATIBILITY.md). Then open **Settings → Diagnostics & compatibility**, run the local checks, and review the generated `summary.txt` and `diagnostics.json` before exporting. The report intentionally excludes chat content, DOM text, usernames, complete local paths, and page titles; attaching it to GitHub remains a deliberate user action.

Use the issue form that matches the problem:

- Windows install, discovery, restart, or lifecycle evidence: **Windows compatibility report**.
- Imported or exported companion validation and rendering problems: **Companion package bug**.
- A proposed built-in or reusable companion: **Companion proposal**.
- Theme, update, or general runtime problems: the general bug form.
