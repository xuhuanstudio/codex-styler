# Roadmap

The roadmap is organized by release gates, not calendar promises.

## Foundation — implemented

- Tauri 2, React, TypeScript, and Rust workspace
- public theme v1 schema and archive validation
- desktop information architecture and bilingual UI
- original icon, backgrounds, and 16-direction companion
- Astro bilingual documentation and SEO metadata
- repository safety, contribution, and release scaffolding
- imported and duplicated theme assets persisted in local WebView application data

## macOS Prototype — implemented, verification continuing

- Codex installation and process detection
- random loopback port and controlled launch
- trusted CDP target filtering
- idempotent safe-mode scene injection
- pause and one-click runtime restore

## macOS Alpha — in progress

- installable Apple Silicon DMG with explicit unnotarized-preview warnings
- SHA-256 checksums, SPDX SBOM, and GitHub build attestations
- adapter fixtures for current Codex versions
- signed theme-package example fixtures
- installation, sleep/wake, crash, upgrade, and uninstall smoke tests

## Cross-platform Beta

- real-device Windows 11 x64 validation
- Windows detection, launch, installer, and restore hardening
- compatibility matrix populated with tested Codex versions
- accessible keyboard and reduced-motion acceptance pass

## v1 Stable

- macOS signing and notarization
- Windows code signing
- signed updater metadata, checksums, and SBOM
- no high-severity dependency findings
- complete restore and rollback verification

## Later, only after evidence

- multi-entity editing
- video backgrounds
- built-in WebGL renderer
- community theme index
- official Pets-compatible asset export
